Manual

This extension provides functions for archiving websites with a cryptographic signature

As a result, it is possible to prove that a datum existed before a given time.

Archiving

In order to archive a page, it has to be open as active browser tab. In the main menu of Firefox you can select the item "Archive". A message should indicate the start of process.

Once a summary is shown, the process finished and was successful. The result file is avilable on your device under 

/mnt/sdcard/
and contains the domain name and a timestamp value (e.g 
example.org-12345678.zip
)

Was there an error message during the process, it is expected that it failed and did not produce a valid output.

Summary

After the archiving process a summary is opened and gives information about the result. It includes the archived URL, a list of ressources, a list of ressources that caused an error and the issuer and the serial number of the certificate used to sign the timestamp.

Validation

Archives that were create with this extension can be verified. By slecting "Verify archive" in the main menu, you are promted to select the archive file.

The process is started automatically and also shows an report once finished. In addition to the report after archiving, some information about the trust status of the certificate is shown.

Hashes match

Do the hash values match, the archive was not altered since the archiving process.

Certificate is trusted

If a certificate is trusted, the signature was created with a certificate that was labeled as truxted by Firefox.

A certificate can be untrusted when it is expired or revoked. In this case it has to be ensured the signature was create in the valid period or before the revocation date. You may contact a expert in this case.

Signatur is correct

If the signature is correct, it was created by the certificate included in the response.

Moving archives

The archives saved on a device can be moved to a computer or a different mobile device like any other file.

USB

By using a USB cable between your device and a computer, files can be compied by using a file browser.

Network connection

By using a filebrowser on your mobile device, the archives can be sent using different methods, for instance E-Mail.

Configuration

The preferences of this extension can be changed in the Firefox Add-On Manager. There you can change the address of the timestamp server.

Any server can be used that implements RFC 3161 with RSA.

Intermediate certificates

Please note that not all timestamp servers include all required certificates to generate a trusted certification chain.

You may need to install additional certificates in Firefox before they can be used. Instructions can usually be found on the website of the server operator.
